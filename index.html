Senior Computer Scientist

Chunhua "Leo" Liao

Center for Applied Scientific Computing (CASC)
Lawrence Livermore National Laboratory

I work at the intersection of compiler technologies, high-performance computing (HPC), and AI-assisted programming tools. My research focuses on making parallel programs more correct, performant, and accessible — from static analysis and automated parallelization to leveraging large language models for code transformation and optimization.

Research Interests

Bridging classical compiler theory with modern AI to advance high-performance scientific computing.

🧩

Compiler Technologies

Developing advanced compiler infrastructure (ROSE) for source-to-source transformation, static analysis, and automated optimization of large-scale scientific codes.

⚡

Parallel Programming Models

Advancing OpenMP and other parallel programming abstractions — enabling automated parallelization, race detection, and performance tuning for HPC workloads.

🤖

AI-Assisted Programming

Leveraging Large Language Models (LLMs) for code optimization, Fortran-to-C++ migration, compiler report interpretation, and security intelligence — CompilerGPT, HPC-GPT, Fortran2CPP.

🔍

Correctness & Verification

Building benchmark suites (DataRaceBench, AutoParBench) and analysis tools to detect data races, validate auto-parallelization, and ensure correctness of parallel programs.

🛡️

Security Intelligence

Applying LLM-driven document clustering and NLP to cyber-security intelligence — automating threat analysis and information extraction from unstructured data.

📊

HPC Performance Optimization

Combining compiler analysis with AI to guide performance engineers — translating cryptic compiler reports into actionable insights for real-world scientific applications.
